Acerca de CreativeSolution

Soluciones creativas de software

CreativeSolution es un equipo de Freelancers dispuestos a desarrollar soluciones ajustadas a un presupuesto para ideas, problemas de tu negocio, comunidad o incluso familiares o personales mediante software:

  • Aplicaciónes móviles
  • Aplicaciones web
  • Páginas web
  • Programas de escritorio
  • Y muchas más

¿Cómo funciona nuestro servicio?

1) Crea una cuenta

Para crear una cuenta solo necesitarás un correo elecrónico y servirá para comunicarte con nosotros y hacer uso de nuestro servicio

2) Descríbenos qué necesitas

Cuéntanos detalladamente tu idea, problema o ayuda que necesites solucionar mediante software.

3) Ingresa el presupuesto

Ingresa el presupuesto dispuesto a invertir en nuestra solución, sin importar lo alto o bajo que sea desarrollaremos una propuesta creativa que se ajuste a él.

4) Revisa nuestra solución

Te enviaremos una propuesta de solución creativa ajustada al presupuesto, puedes revisarla, aceptarla, proponer modificaciones o rechazarla.

5) Obtén un demo

Una vez acordemos una solución, crearemos un demo sin ningún costo y te lo enviaremos para que lo revices.

6) Obtén la solución

Si el demo te convence, pactaremos el precio final, la forma de pago, las fechas de entrega y el método de trabajo para crear la solución completa.
